## 執行專案

1.執行終端機並下載專案

```
git clone https://github.com/Lan956616/ALPHA-Shop-II.git
```

2.切換至專案資料夾

```
cd ALPHA-Shop-II
```

3.安裝 npm 套件

```
npm install
```

4.執行網頁

```
npm run dev
```

5.暫停執行網頁

```
ctrl +　ｃ
```
